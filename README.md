# make-a-call-app

## Stack used
```
>> nodejs
>> expressjs
>> postgres
>> Sequelize ORM
>> plivo service (for voice call)
```

#### Install postgres and create database named "make_call_db" with user "postgres" and password "postgres"



## Steps to Run

### Run back end app

```
>> cd server
>> npm install
>> npm run start
```

### Run front end app

```
>> cd client
>> npm install
>> npm start
```
